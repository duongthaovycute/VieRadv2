Core Features

Tính năng chính

01

Dashboard tối giản

Vòng tròn lớn hiển thị số liệu chính. Bấm vào vòng tròn để chuyển giữa suất liều, CPS, CPM, Total, nhiệt độ và độ ẩm.

02

Nhiều trạm đo

Hiển thị từng trạm riêng, có trạng thái online, mất kết nối, nghỉ hoặc cảnh báo.

03

Cảnh báo màu

Đổi màu theo ngưỡng: bình thường, chú ý, cao và vượt ngưỡng. Người dùng nhìn màu là hiểu ngay.

04

API / ReWES / Bluetooth

Hỗ trợ lấy dữ liệu qua API, ReWES và Bluetooth LE. Có phần nhập URL hoặc cấu hình thiết bị.

05

Biểu đồ phân tích

Có biểu đồ theo thời gian cho từng trạm và biểu đồ tổng để so sánh nhiều nguồn dữ liệu.

06

Xuất báo cáo

Hỗ trợ xuất dữ liệu, tạo báo cáo và gửi qua email nếu người dùng đã đăng nhập.

07

Cài đặt giao diện

Đổi màu chủ đạo, sáng/tối, ngôn ngữ, ngưỡng cảnh báo và tốc độ cập nhật.

08

Dễ bảo trì

Web tách file cấu hình riêng nên đổi link APK, phiên bản, GitHub, email không cần sửa nhiều.

App Flow

Luồng giao diện được đề xuất

Các trang chính được chia rõ để trang đầu luôn nhẹ, đẹp và không bị rối.

1

Tổng quan

Số liệu chính trong vòng tròn, chọn trạm, cảnh báo màu, thông tin phụ ở góc.

2

Tất cả trạm

Bảng nhiều trạm, lọc trạng thái, xem nhanh trạm nào đang cảnh báo hoặc mất kết nối.

3

Biểu đồ

Biểu đồ từng trạm và biểu đồ tổng để theo dõi xu hướng dữ liệu.

4

Kết nối

Cấu hình API, ReWES, Bluetooth LE và custom endpoint.

5

Báo cáo

Xuất CSV/PDF, copy báo cáo, gửi email khi đã đăng nhập.

6

Cài đặt

Đổi màu, dark mode, ngưỡng cảnh báo, ngôn ngữ và thông tin app.

User Guide

Hướng dẫn sử dụng nhanh

01

Tải APK

Bấm “Tải APK về điện thoại” ở khu vực Download Center.

02

Cài đặt

Mở file APK vừa tải. Nếu bị chặn, bật quyền cài app từ trình duyệt.

03

Chọn trạm

Vào Tổng quan, chọn trạm cần theo dõi hoặc cấu hình API/Bluetooth.

04

Theo dõi cảnh báo

Quan sát màu vòng tròn và trạng thái trạm để biết mức dữ liệu hiện tại.

05

Xem biểu đồ

Mở trang Biểu đồ để xem xu hướng theo thời gian và so sánh nhiều trạm.

06

Xuất báo cáo

Chọn khoảng thời gian, loại dữ liệu và xuất báo cáo khi cần lưu trữ.

FAQ

Câu hỏi thường gặp

Web này có phải app không?

Không. Đây là website giới thiệu, hướng dẫn và cho tải . App chính vẫn là VieRad cài trên điện thoại Android.

Có thể gắn API thật lên web không?

Có. Web hiện có phần demo. Sau này có thể nối API thật để hiển thị số liệu trực tiếp ngay trên website.

Web có bị cache link cũ không?

Bản này tách cấu hình và có version query ở CSS/JS để giảm lỗi cache. Khi đổi lớn, nên đổi số version trong link file.
